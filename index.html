Collab is a Django project with a standard set of configurations to provide services to reusable apps.

Repositories

  • collab

    Collab core repo

Features

Core Collab comes with:

  • Tagging

  • Notifications

  • Widgets

  • Search

  • Migrations

  • Improved Caching

  • Javascript Libraries
